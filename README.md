This app built using eclipse IDE.
Includes 3 API without DB(static course array)
  1. GET Courses API
  2. GET Course API by courseId
  3. POST addCourse API by course

Created basic application using start.spring.io.

localhost:8080/courses - GET
localhost:8080/courses/1 - GET
localhost:8080/courses - POST
